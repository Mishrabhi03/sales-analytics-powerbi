# Sales Analytics Dashboard (Power BI)

## Overview
This project is an end-to-end sales analytics dashboard built using SQL and Power BI.
It analyzes revenue trends, category-wise profitability, and regional performance to
support data-driven business decisions.

## Tools & Technologies
- SQL (data cleaning and aggregation)
- Power BI (data modeling and visualization)
- Excel / CSV (data export)
- Mock AI insights for executive summary

## Key Features
- Total Revenue KPI
- Monthly Sales Trend analysis
- Category-wise Revenue vs Profit comparison
- Region-wise Revenue performance
- Interactive slicers for category filtering
- AI-style insights derived from data patterns

## Business Insights
- Sales show a steady upward trend indicating consistent demand growth
- Technology is the strongest revenue contributor
- Furniture has lower profit margins and requires strategy review
- High-performing regions should be prioritized for growth
